Linear Algebra Review

Today we review the essential linear algebra concepts needed for this course. Even if students have seen this before, it's worth revisiting with an optimization perspective.

Vectors and Matrices

  • Vector spaces
  • Linear transformations
  • Matrix operations
Start with the definition of a vector space. Emphasize that we work primarily in R^n but the concepts generalize. Linear transformations preserve vector space structure - this is key for understanding why matrices are so useful.

Eigenvalues

For a matrix \(A\), eigenvalue \(\lambda\) satisfies:

\[Av = \lambda v\]

Eigenvalues tell us about the "stretching" behavior of a matrix. For symmetric matrices, eigenvalues are real. The largest eigenvalue determines the spectral radius. We'll use eigenvalues extensively when analyzing convergence of iterative methods.

Applications

  • Principal Component Analysis
  • Spectral clustering
  • Graph algorithms
PCA uses eigendecomposition to find directions of maximum variance. Spectral clustering uses eigenvectors of the graph Laplacian. PageRank is essentially finding the dominant eigenvector of a stochastic matrix.
