Interior Point Methods

Interior point methods revolutionized optimization in the 1980s. They can solve linear programs in polynomial time and are the method of choice for many structured convex problems. The key idea: stay in the interior of the feasible region and use barriers to handle constraints.

Barrier Functions

Log barrier:

\[\phi(x) = -\sum_{i=1}^m \log(-g_i(x))\]

The log barrier goes to infinity as we approach the constraint boundary. This keeps iterates strictly feasible. We add t * phi(x) to the objective, where t controls the barrier strength. As t increases, the solution approaches the true optimum.

Central Path

  • Path following methods
  • Newton steps
  • Complexity analysis
The central path is the set of minimizers as t varies. We follow this path using Newton's method. Each Newton step is expensive (requires solving a linear system) but very effective. Total complexity is O(sqrt(m) log(1/epsilon)) Newton steps for m constraints - polynomial in all parameters!

Applications

  • Linear programming
  • Semidefinite programming
  • Quadratic programming
LP solvers like CPLEX and Gurobi use interior point methods alongside simplex. SDP is crucial for relaxations in combinatorial optimization and control theory. QP appears in portfolio optimization, model predictive control, and support vector machines.
