Convex Optimization

Convexity is the single most important concept in optimization. Convex problems have the remarkable property that any local minimum is also a global minimum. This makes them tractable both theoretically and computationally.

Convex Sets

A set \(C\) is convex if:

\[\forall x, y \in C, \forall \theta \in [0,1]: \theta x + (1-\theta)y \in C\]

Draw pictures! A set is convex if the line segment between any two points in the set stays in the set. Examples: balls, halfspaces, polyhedra. Non-examples: donuts, stars. Intersection of convex sets is convex - this is very useful.

Convex Functions

  • Definition and properties
  • First-order conditions
  • Second-order conditions
A function is convex if its epigraph is a convex set. Equivalently, the function lies below any chord. First-order condition: f(y) >= f(x) + gradient dot (y-x). Second-order: Hessian is positive semidefinite. Common examples: norms, quadratics with PSD matrix, log-sum-exp.

Optimization Problems

\[\min_{x} f(x) \quad \text{s.t.} \quad g_i(x) \leq 0, \; h_j(x) = 0\]

A problem is convex if: objective is convex, inequality constraints are convex, equality constraints are affine. The feasible set is then convex. KKT conditions are necessary AND sufficient for convex problems. This is why convex optimization is so powerful.
